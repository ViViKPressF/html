    • Основные HTML-коды для создания веб-страницы

    1. Базовая структура HTML-документа 

    <!DOCTYPE html>
  <html lang="ru">
  <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Название страницы</title>
  </head>
  <body>
      <!-- Содержимое страницы -->
  </body>
  </html>

    2. Заголовки и текстовые элементы 

    <h1>Главный заголовок</h1>
<h2>Заголовок второго уровня</h2>
<h3>Заголовок третьего уровня</h3>

<p>Это абзац текста.</p>
<p>Еще один абзац с <strong>жирным</strong> и <em>курсивным</em> текстом.</p>

    3.Ссылки и изображения 

    
<a href="https://example.com">Текст ссылки</a>
<a href="page.html">Ссылка на другую страницу</a>
<a href="#section">Якорная ссылка</a>

<img src="image.jpg" alt="Описание изображения" width="300">

    4. Списки

    <!-- Маркированный список -->
<ul>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
</ul>

<!-- Нумерованный список -->
<ol>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
</ol>

    5. Таблица 

    <table border="1">
<tr>
    <th>Заголовок 1</th>
    <th>Заголовок 2</th>
</tr>
<tr>
    <td>Ячейка 1</td>
    <td>Ячейка 2</td>
</tr>
</table>

    6. Форма

    <form action="/submit" method="post">
<label for="name">Имя:</label>
<input type="text" id="name" name="name"><br>

<label for="email">Email:</label>
<input type="email" id="email" name="email"><br>

<input type="submit" value="Отправить">
</form>

    7. Семантические элементы

    <header>Шапка сайта</header>
<nav>Навигация</nav>
<main>
    <article>Статья</article>
    <section>Раздел</section>
</main>
<aside>Боковая панель</aside>
<footer>Подвал</footer>
